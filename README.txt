# Платформер в стиле Марио

Простая 2D-игра-платформер с параллаксной прокруткой и анимацией персонажей.

## Особенности

- Фоновый эффект параллакса
- Плавное перемещение персонажа и прыжки
- Столкновения с платформой
- Адаптивный дизайн для мобильных устройств и компьютеров
- Клавиатура и сенсорное управление
- Имеюстя противники
- Возможность собрать монеты
- Система уровней

## Как запустить

1. Копируем файлы куда удобно
2. Открываем `index.html" в веб-браузере
 2.1. Переходим по ссылке
3. Используем клавиши со стрелками для перемещения и пробел для прыжка

## Элементы управления

- ** Рабочий стол **: Клавиши со стрелками для перемещения, пробел для прыжка
- ** Мобильный **: Коснитесь левой стороны, чтобы переместиться влево, правой стороны, чтобы переместиться вправо, любое касание для перехода

## Используемые технологии

- HTML5 Canvas
- Простой JavaScript
- CSS для адаптивного дизайна
